# Electric Vehicle Sales Analysis – India 🚗⚡

## 📌 Objective
To analyze electric vehicle sales trends in India using Python, SQL, Excel, and Machine Learning.

## 📊 Dataset
Electric Vehicle Sales by State in India (CSV format)

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib, Scikit-learn)
- SQL (SQLite / MySQL)
- Excel (Dashboard & Pivot Tables)
- Machine Learning (Random Forest Regressor)

## 🔍 Project Workflow
1. Data cleaning & preprocessing
2. Feature engineering (Month, Day extraction)
3. Exploratory Data Analysis (EDA)
4. One-hot encoding
5. Machine Learning model training
6. SQL-based analysis
7. Excel dashboard creation

## 📈 Key Insights
- EV sales have increased year-over-year
- Two-wheelers dominate EV sales
- Maharashtra and Karnataka are leading states

## ▶️ How to Run
1. Clone the repository
2. Run the Jupyter Notebook (`.ipynb`)
3. Open Excel dashboard for visualization
